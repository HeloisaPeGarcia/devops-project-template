# Política de Segurança (Security Policy)

## Versões Suportadas

Atualmente, apenas a versão mais recente deste template recebe atualizações de segurança.

| Versão | Suportada |
| ------- | --------- |
| 1.0.x   | ✅ Sim     |
| < 1.0   | ❌ Não     |

## Relatando uma Vulnerabilidade

Levamos a segurança deste projeto a sério. Se você descobrir uma vulnerabilidade de segurança, siga os passos abaixo para reportá-la de forma responsável:

1. **Não abra uma issue pública** para relatar problemas de segurança.
2. Envie um e-mail para `seguranca@exemplo.com` com uma descrição detalhada da vulnerabilidade, etapas para reproduzir e possíveis impactos.
3. Entraremos em contato em até 48 horas para confirmar o recebimento e planejar a correção.
